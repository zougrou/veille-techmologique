Mesure de prevention


Les mesure de prevention sont les suivantes:
1- Restez chez vous autant que possible.
2- GARDEZ une distance de sécurité.
3- LAVEZ-VOUS souvant les mains.
4- COUVRE-VOUS la bouche quand vous toussez.
5- VOUS ETRES MALADE? Appelez votre médecin.
